# 🥦 Gestión AFV — Abarrotes, Frutas y Verduras

Sistema de gestión para negocios minoristas de abarrotes, frutas y verduras.

- Punto de venta (POS) para cajeras
- Gestión de stock en tiempo real
- Control de ingresos y ventas
- Roles: Cajera (solo ventas diarias) y Admin (todo el sistema)

Desarrollado con PHP + MySQL + Railway.

---

## 🔧 Requisitos

- PHP 8.0+
- MySQL 8.0+
- Composer (para dependencias futuras)

## 🚀 Despliegue

Este proyecto está diseñado para desplegarse en **Railway.app**.

1. Crea un nuevo proyecto en Railway
2. Conéctalo a este repositorio
3. Railway detectará automáticamente `railway.toml` y `public/`
